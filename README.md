# captcha_recognition
Tensorflow2 简单验证码识别 使用卷积神经网络

## 说明
- python模块版本
```
Pillow==7.0.0
tensorflow==2.1.0
matplotlib==3.1.3
pandas==1.0.2
```

- simple.h5为训练好的模型
